# Booby Tracker

A lightweight browser-based nursing tracker for timing left/right feeds, logging pumped milk, and viewing daily/weekly/monthly stats.

## How to install

iPhone: Open in Safari → Share button → Add to Home Screen
Android: Open in Chrome → three-dot menu → Add to Home Screen (or Chrome will prompt automatically)

## Features

- Start/stop left or right nursing sessions with one tap
- Log pumped milk volume in milliliters
- View recent feed entries and manage logs
- Filter logs by side or date range
- Summary tabs for today, week, and month
- Stores data locally in the browser using `localStorage`

## Usage

1. Open `index.html` in a browser.
2. Tap `Left` or `Right` to start timing a feed.
3. Tap the same side again to stop and save the session.
4. Use `Log` to review or delete entries.
5. Use `Stats` to see feed totals, average duration, and pumping volume.

## Notes

- No backend required.
- Data remains in the browser until local storage is cleared.
